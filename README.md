### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bishtayush1017-spec/flutter_chat_app.git
   cd flutter_chat_app
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```
